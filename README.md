# ✊ Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript. The player selects Rock, Paper, or Scissors, and the computer randomly generates its choice. The game then displays the result based on both selections.

🚀 Features

* Play Rock Paper Scissors against the computer
* Random computer choice generation
* Instant result display
* Clean and responsive user interface
* Built with pure HTML, CSS, and JavaScript

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript


🎮 How to Play

1. Open the game in your browser.
2. Choose:

   * Rock ✊
   * Paper ✋
   * Scissors ✌️
3. The computer will randomly select its choice.
4. The game will display:

   * Result (Win, Lose, or Draw)

🏆 Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Matching choices result in a Draw

💻 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
